# wiretide-controller
Controller and agent for OpenWRT devices

Install on Ubuntu 24.04

wget https://raw.githubusercontent.com/simonsays-techtalk/wiretide-controller/main/install.sh -O install.sh
chmod +x install.sh
sudo ./install.sh
